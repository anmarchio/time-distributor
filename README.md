# Time Distributor

Small Flask + JavaScript tool to distribute weekly working hours across project tickets using weighted allocation.

## Features

- Weekly work time input
- Ticket management
- Weighted time distribution (1–5)
- Randomized but realistic allocation
- CSV export

## Start

Install Python 3 first.

Run:

```bash
pip install flask
python app.py
```

or simply double-click: `start_time_distributor.bat`

Then open: `http://127.0.0.1:5000`
